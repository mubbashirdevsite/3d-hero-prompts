Pro tips

Get better results

Performance
Always set a pixel ratio cap
Add "cap devicePixelRatio at 2 for performance" to any Three.js prompt. High-DPI displays can halve your frame rate on heavy scenes.
Accessibility
Request a reduced-motion fallback
Append "add a prefers-reduced-motion media query that stops all animation" to every prompt to avoid excluding users.
Mobile
Ask for mobile particle reduction
Add "halve particle count on screens narrower than 768 px" — mobile GPUs can't handle the same density as desktop.
